# Productivity Dashboard

A modern and responsive Productivity Dashboard built using HTML, CSS, and JavaScript. The application helps users organize their daily work with multiple productivity tools in one place. It features a clean UI, smooth theme switching, and an interactive user experience.

# Features

 • To-Do List
  - Add and delete tasks
  - Mark tasks as completed
  - Filter tasks by All, Completed, and Pending

 • Daily Planner
  - Schedule activities with time slots
  - Organize your daily routine

 • Pomodoro Timer
  - 25-minute focus session
  - Start, Pause, and Reset controls

 • Goals Tracker
  - Add and manage personal goals

 • Weather
  - Search weather by city
  - Displays current weather information using a Weather API

 • Daily Motivation
  - Fetches inspirational quotes using the API Ninjas Quotes API
  - Copy quotes to clipboard

 • Light & Dark Theme
  - Smooth theme switching
  - Implemented using CSS Root Variables

 • Modern UI
  - Responsive design
  - Custom cursor glow effect
  - Smooth transitions and animations

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- CSS Variables
- Local Storage
